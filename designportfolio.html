Some projects were bigger than others; for example, "Things I'll Never Say" wanted an individual poster for each actor (for personal social media uses) on top of the group publicity photos!

WEBSITE DESIGN

Other than this website, which I wrote the code for using the hosting services of GitHub, I've also designed websites both for freelance work and as part of my previous employment.

If you'd like an example of my freelance work, go to Goal-Tender. Nikki Shapiro, Principal talent at Goal-Tender, says about my work:
"She was very knowledgable about all aspects of website design, and offered to assist me above and beyond the original parameters of the contract. She was pleasant and available and timely."

At Wesleyan University Press, I designed graphics for their website and blog (like the blog header, which they still use!) as well as webpages using Wordpress.com. Just a few of those were Reader's Companion pages, like these for Gerald Vizenor, Fred Moten, Sarah Blake, Ben Doller, Rae Armentrout, Heather Christle, Honorée Fanonne Jeffers, M. NourbeSe Philip, and Brenda Hillman. The base website is created using Cascade, so I also gained practice on that software as well; for example, I wrote the copy and designed all elements of this page for Best Experimental Writing 2016 Anthology.
